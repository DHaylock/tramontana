Tramontana

What is Tramontana?

Tramonatana is intended as a tool for designers and creatives to use iPhones and iPads as sensors or actuators and create quick interactive prototypes of apps, interactive spaces and objects.

Tramontana is a set of libraries and iOS and tvOS apps. Tramontana app exposes access to all the features of your phone from the open source javascript or openFrameworks libraries.

You can use your device as actuator triggering from a webpage or an openframeworks sketch running on a computer or different device. The features you can access are:
• flash light;
• vibration;
• changes to screen brightness and color:
• playback of video, audio or images;

Alternatively you can use the device as a sensor reading the device:
• orientation;
• magnetometer;
• accelerometer;
• distance sensor;
• audio jack;
• changes in power source;
• touch events;

Getting started

To get started select your preferred platform, Javascript, openFrameworks or Processing and follow the instructions on the plugin or libraries page.

javascript

Please refer to the javascript github page for more informations: javascript

            var iphone=new BFtObject();
            iphone.start('192.168.1.2');
            iphone.makeVibrate();
openFrameworks

The openFrameworks plugin contains examples that show how you can use your phone as sensor or actuator. More info on the plugin page: openFrameworks
